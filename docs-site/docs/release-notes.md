# Release Notes

Assist Agent Sandbox is currently documented as an early `0.1.0` preview. Use the repository changelog as the source of truth for release history:

- [`CHANGELOG.md`](https://github.com/bradsjm/hacs-llm-sandbox/blob/main/CHANGELOG.md)

## Preview guidance

Because this is preview software, review changes carefully before enabling service actions. Re-test read-only prompts after each upgrade, then test action prompts only if your action settings are intentionally enabled.
