# Monty Globals

The exact prompt surface is generated by the integration, but the practical API centers on these globals and facades.

| Global or facade | Purpose |
| --- | --- |
| `hass` | Safe Home Assistant facade with `states`, `services`, `config`, `history`, and `query`. |
| `states` | Convenience access to the safe state machine. |
| `er`, `dr`, `ar`, `fr`, `lr` | Registry-style facades for entity, device, area, floor, and label access. |
| `json`, `math`, `re` | Supported imports documented by the README. |

## Common code shapes

```python
result = [s.entity_id for s in hass.states.async_all() if s.domain == "light" and s.state == "on"]
```

```python
result = await hass.query(
    "select entity_id, state from states where domain = 'sensor' limit 20",
    hours=1,
)
```
